# Christmas Landing (Landing)

A minimal React + TypeScript + Tailwind landing page for "Christmas Wonderland 2025".

Scripts:
- pnpm install
- pnpm dev

This project demonstrates:
- Hero (with falling snow CSS)
- Countdown (updates every second)
- Gift Grid (6 cards with hover scale)
- Footer with social icons

Theme colors: Red (#C53030), Green (#2F855A), Gold (#D69E2E)

Test Cases:
- Project bootstrapping: pnpm install & pnpm dev
- UI rendering: all components render, no console errors
- Responsiveness: mobile/tablet/desktop using Tailwind breakpoints
