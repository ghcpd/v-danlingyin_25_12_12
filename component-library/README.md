# Christmas UI Component Library (Demo)

A small collection of reusable components (Button, Card, Badge) with a demo page.

Scripts:
- pnpm install
- pnpm dev

This package demonstrates:
- Reusable Button (primary/ghost)
- Card component
- Badge (red/gold)

Test cases:
- Bootstrapping: pnpm install & pnpm dev
- Rendering: components mount with no errors
- Responsiveness: components scale across breakpoints
